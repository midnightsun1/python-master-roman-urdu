# 🐍 Python Master — Roman Urdu

Roman Urdu mein Python seekhne ki **complete interactive app**.

## Install Karo

```bash
pip install python-master-roman-urdu
```

## Run Karo

```bash
python-master
```

Ya directly:

```bash
python -m python_master.cli
```

## Features

- 🌱 Beginner se Advanced tak complete curriculum
- 💻 Built-in code runner — browser mein hi code likho aur chalao
- 📋 Python Cheat Sheet
- 🛠️ Practice Projects
- 🤖 AI/ML Roadmap
- ❓ Har lesson ke baad Quiz

## Requirements

- Python 3.8+
- streamli >= 1.28.0
🐍 Python Master - Roman Urdu Edition
Is repository ka maqsad un logon ki help karna hai jo Python programming seekhna chahte hain lekin English ki wajah se mushkil mehsoos karte hain. Humne coding ke concepts ko Roman Urdu mein bohot hi asaan alfaaz mein samjhaya hai.

🌟 Features
Asaan Zubaan: Coding ke mushkil concepts ko rozmara ki Roman Urdu mein explain kiya gaya hai.

Beginner Friendly: Agar aapne aaj tak code nahi likha, tab bhi yeh course aapke liye hai.

Structured Content: Basic se lekar intermediate level tak ke topics.

📂 Content Index
Yahan par aapko ye topics milenge:

Introduction: Python kya hai aur ise install kaise karein?

Variables & Data Types: Data ko memory mein kaise store karte hain?

Conditions (If/Else): Program se faislay (decisions) kaise karwayein?

Loops (For/While): Kaam ko bar bar automatic kaise karein?

Functions: Code ko reuse kaise karein?

(Aap yahan mazeed topics add kar sakte hain...)

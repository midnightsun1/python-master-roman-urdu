# 🐍 Python Master — Roman Urdu

Roman Urdu mein Python seekhne ki **complete interactive app**. Is project ka maqsad un logon ki help karna hai jo Python programming seekhne mein mushkil mehsoos karte hain.

## 🛠️ Requirements

* **Python:** 3.8+
* **Streamlit:** >= 1.28.0

## 📥 Kaise Install Karein?

Apne terminal ya command prompt mein yeh command chalayein:

```bash
pip install python-master-roman-urdu
```

## 🚀 Kaise Run Karein?

App ko start karne ke liye:

```bash
python-master
```

Ya phir direct module ke zariye:

```bash
python -m python_master.cli
```

## ✨ Features

* 🌱 **Beginner to Advanced** — Complete curriculum basic se professional level tak.
